sequencer
=========

This sequencer is very simple.

<キーボード操作一覧>

P :タイムカウントが始まります。

M :タイムカウントが一時停止します。

R :スイッチ？がリセットされます。

↑：タイムカウントのスピードが上がります。

↓：タイムカウントのスピードが下がります。


<SAVE機能について>

・最大5つまでのスイッチの情報が保存されます。

・保存されたスイッチの情報はウィンドウの右側に画像として表示されます。

・保存の仕方は保存したいときにF1〜F5のいずれかを押して保存します。

・上からF1、F2、F3、F4、F5と保存されたスイッチの情報が表示されます。

・画像をマウスでクリックすると保存されたスイッチの情報を読み込みます。
